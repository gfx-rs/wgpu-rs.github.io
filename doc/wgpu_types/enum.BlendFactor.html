Expand description

Alpha blend factor.

Corresponds to WebGPU GPUBlendFactor. Values using Src1 require Features::DUAL_SOURCE_BLENDING and can only be used with the first render target.

For further details on how the blend factors are applied, see the analogous functionality in OpenGL: https://www.khronos.org/opengl/wiki/Blending#Blending_Parameters.
