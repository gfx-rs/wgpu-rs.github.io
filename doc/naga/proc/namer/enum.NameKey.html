FunctionOobLocal(Handle<Function>, Handle<Type>)

A local variable used by ReadZeroSkipWrite bounds-check policy when it needs to produce a pointer-typed result for an OOB access. These are unique per accessed type, so the second element is a type handle. See docs for crate::back::msl.
