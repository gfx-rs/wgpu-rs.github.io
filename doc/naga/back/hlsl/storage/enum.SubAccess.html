Enum SubAccess

pub(super) enum SubAccess {
    BufferOffset {
        group: u32,

        value: Handle<Expression>,
        stride: u32,
    },
}
Expand description

One step in accessing a Storage global’s component or element.

Writer::temp_access_chain holds a series of these structures, describing how to compute the byte offset of a particular element or member of some global variable in the Storage address space.

Fields

§

Offset(u32)

Add the given byte offset. This is used for struct members, or known components of a vector or matrix. In all those cases, the byte offset is a compile-time constant.

§

Index

Scale value by stride, and add that to the current byte offset. This is used to compute the offset of an array element whose index is computed at runtime.
