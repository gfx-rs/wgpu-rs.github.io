Struct ErrorFuture

Source 
struct ErrorFuture<F> {
    inner: F,
}
Expand description

A wrapper for pop_error_scope futures that panics if an error occurs.

Given a future inner of an Option<E> for some error type E, wait for the future to be ready, and panic if its value is Some.

This can be done simpler with FutureExt, but we don’t want to add a dependency just for this small case.
