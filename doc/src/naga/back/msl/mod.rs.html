1/*!
2Backend for [MSL][msl] (Metal Shading Language).
3
4This backend does not support the [`SHADER_INT64_ATOMIC_ALL_OPS`][all-atom]
5capability.
6
7## Binding model
8
9Metal's bindings are flat per resource. Since there isn't an obvious mapping
10from SPIR-V's descriptor sets, we require a separate mapping provided in the options.
11This mapping may have one or more resource end points for each descriptor set + index
12pair.
13
14## Entry points
15
16Even though MSL and our IR appear to be similar in that the entry points in both can
17accept arguments and return values, the restrictions are different.
18MSL allows the varyings to be either in separate arguments, or inside a single
19`[[stage_in]]` struct. We gather input varyings and form this artificial structure.
20We also add all the (non-Private) globals into the arguments.
21
22At the beginning of the entry point, we assign the local constants and re-compose
23the arguments as they are declared on IR side, so that the rest of the logic can
24pretend that MSL doesn't have all the restrictions it has.
25
26For the result type, if it's a structure, we re-compose it with a temporary value
27holding the result.
28
29[msl]: https://developer.apple.com/metal/Metal-Shading-Language-Specification.pdf
30[all-atom]: crate::valid::Capabilities::SHADER_INT64_ATOMIC_ALL_OPS
31
32## Pointer-typed bounds-checked expressions and OOB locals
33
34MSL (unlike HLSL and GLSL) has native support for pointer-typed function
35arguments. When the [`BoundsCheckPolicy`] is `ReadZeroSkipWrite` and an
36out-of-bounds index expression is used for such an argument, our strategy is to
37pass a pointer to a dummy variable. These dummy variables are called "OOB
38locals". We emit at most one OOB local per function for each type, since all
39expressions producing a result of that type can share the same OOB local. (Note
40that the OOB local mechanism is not actually implementing "skip write", nor even
41"read zero" in some cases of read-after-write, but doing so would require
42additional effort and the difference is unlikely to matter.)
43
44[`BoundsCheckPolicy`]: crate::proc::BoundsCheckPolicy
45
46## External textures
47
48Support for [`crate::ImageClass::External`] textures is implemented by lowering
49each external texture global variable to 3 `texture2d<float, sample>`s, and a
50constant buffer of type `NagaExternalTextureParams`. This provides up to 3
51planes of texture data (for example single planar RGBA, or separate Y, Cb, and
52Cr planes), and the parameters buffer containing information describing how to
53handle these correctly. The bind target to use for each of these globals is
54specified via the [`BindTarget::external_texture`] field of the relevant
55entries in [`EntryPointResources::resources`].
56
57External textures are supported by WGSL's `textureDimensions()`,
58`textureLoad()`, and `textureSampleBaseClampToEdge()` built-in functions. These
59are implemented using helper functions. See the following functions for how
60these are generated:
61 * `Writer::write_wrapped_image_query`
62 * `Writer::write_wrapped_image_load`
63 * `Writer::write_wrapped_image_sample`
64
65The lowered global variables for each external texture global are passed to the
66entry point as separate arguments (see "Entry points" above). However, they are
67then wrapped in a struct to allow them to be conveniently passed to user
68defined and helper functions. See `writer::EXTERNAL_TEXTURE_WRAPPER_STRUCT`.
69*/

131fn deserialize_binding_map<'de, D>(deserializer: D) -> Result<BindingMap, D::Error>
132where
133    D: serde::Deserializer<'de>,
134{
135    use serde::Deserialize;
136
137    let vec = Vec::<BindingMapSerialization>::deserialize(deserializer)?;
138    let mut map = BindingMap::default();
139    for item in vec {
140        map.insert(item.resource_binding, item.bind_target);
141    }
142    Ok(map)
143}

503impl Options {
504    fn resolve_local_binding(
505        &self,
506        binding: &crate::Binding,
507        mode: LocationMode,
508    ) -> Result<ResolvedBinding, Error> {
509        match *binding {
510            crate::Binding::BuiltIn(mut built_in) => {
511                match built_in {
512                    crate::BuiltIn::Position { ref mut invariant } => {
513                        if *invariant && self.lang_version < (2, 1) {
514                            return Err(Error::UnsupportedAttribute("invariant".to_string()));
515                        }
516
517                        // The 'invariant' attribute may only appear on vertex
518                        // shader outputs, not fragment shader inputs.
519                        if !matches!(mode, LocationMode::VertexOutput) {
520                            *invariant = false;
521                        }
522                    }
523                    crate::BuiltIn::BaseInstance if self.lang_version < (1, 2) => {
524                        return Err(Error::UnsupportedAttribute("base_instance".to_string()));
525                    }
526                    crate::BuiltIn::InstanceIndex if self.lang_version < (1, 2) => {
527                        return Err(Error::UnsupportedAttribute("instance_id".to_string()));
528                    }
529                    // macOS: Since Metal 2.2
530                    // iOS: Since Metal 2.3 (check depends on https://github.com/gfx-rs/naga/issues/2164)
531                    crate::BuiltIn::PrimitiveIndex if self.lang_version < (2, 2) => {
532                        return Err(Error::UnsupportedAttribute("primitive_id".to_string()));
533                    }
534                    _ => {}
535                }
536
537                Ok(ResolvedBinding::BuiltIn(built_in))
538            }
539            crate::Binding::Location {
540                location,
541                interpolation,
542                sampling,
543                blend_src,
544            } => match mode {
545                LocationMode::VertexInput => Ok(ResolvedBinding::Attribute(location)),
546                LocationMode::FragmentOutput => {
547                    if blend_src.is_some() && self.lang_version < (1, 2) {
548                        return Err(Error::UnsupportedAttribute("blend_src".to_string()));
549                    }
550                    Ok(ResolvedBinding::Color {
551                        location,
552                        blend_src,
553                    })
554                }
555                LocationMode::VertexOutput | LocationMode::FragmentInput => {
556                    Ok(ResolvedBinding::User {
557                        prefix: if self.spirv_cross_compatibility {
558                            "locn"
559                        } else {
560                            "loc"
561                        },
562                        index: location,
563                        interpolation: {
564                            // unwrap: The verifier ensures that vertex shader outputs and fragment
565                            // shader inputs always have fully specified interpolation, and that
566                            // sampling is `None` only for Flat interpolation.
567                            let interpolation = interpolation.unwrap();
568                            let sampling = sampling.unwrap_or(crate::Sampling::Center);
569                            Some(ResolvedInterpolation::from_binding(interpolation, sampling))
570                        },
571                    })
572                }
573                LocationMode::Uniform => Err(Error::GenericValidation(format!(
574                    "Unexpected Binding::Location({location}) for the Uniform mode"
575                ))),
576            },
577        }
578    }

588    ) -> Option<&BindTarget> {
589        self.get_entry_point_resources(ep)
590            .and_then(|res| res.resources.get(res_binding))
591    }
592
593    fn resolve_resource_binding(
594        &self,
595        ep: &crate::EntryPoint,
596        res_binding: &crate::ResourceBinding,
597    ) -> Result<ResolvedBinding, EntryPointError> {
598        let target = self.get_resource_binding_target(ep, res_binding);
599        match target {
600            Some(target) => Ok(ResolvedBinding::Resource(target.clone())),
601            None if self.fake_missing_bindings => Ok(ResolvedBinding::User {
602                prefix: "fake",
603                index: 0,
604                interpolation: None,
605            }),
606            None => Err(EntryPointError::MissingBindTarget(*res_binding)),
607        }
608    }
609
610    fn resolve_push_constants(
611        &self,
612        ep: &crate::EntryPoint,
613    ) -> Result<ResolvedBinding, EntryPointError> {
614        let slot = self
615            .get_entry_point_resources(ep)
616            .and_then(|res| res.push_constant_buffer);
617        match slot {
618            Some(slot) => Ok(ResolvedBinding::Resource(BindTarget {
619                buffer: Some(slot),
620                ..Default::default()
621            })),
622            None if self.fake_missing_bindings => Ok(ResolvedBinding::User {
623                prefix: "fake",
624                index: 0,
625                interpolation: None,
626            }),
627            None => Err(EntryPointError::MissingPushConstants),
628        }
629    }
630
631    fn resolve_sizes_buffer(
632        &self,
633        ep: &crate::EntryPoint,
634    ) -> Result<ResolvedBinding, EntryPointError> {
635        let slot = self
636            .get_entry_point_resources(ep)
637            .and_then(|res| res.sizes_buffer);
638        match slot {
639            Some(slot) => Ok(ResolvedBinding::Resource(BindTarget {
640                buffer: Some(slot),

642            })),
643            None if self.fake_missing_bindings => Ok(ResolvedBinding::User {
644                prefix: "fake",
645                index: 0,
646                interpolation: None,
647            }),
648            None => Err(EntryPointError::MissingSizesBuffer),
649        }
650    }

653impl ResolvedBinding {
654    fn as_inline_sampler<'a>(&self, options: &'a Options) -> Option<&'a sampler::InlineSampler> {
655        match *self {
656            Self::Resource(BindTarget {
657                sampler: Some(BindSamplerTarget::Inline(index)),
658                ..
659            }) => Some(&options.inline_samplers[index as usize]),
660            _ => None,
661        }
662    }
663
664    fn try_fmt<W: Write>(&self, out: &mut W) -> Result<(), Error> {
665        write!(out, " [[")?;
666        match *self {
667            Self::BuiltIn(built_in) => {
668                use crate::BuiltIn as Bi;
669                let name = match built_in {
670                    Bi::Position { invariant: false } => "position",
671                    Bi::Position { invariant: true } => "position, invariant",
672                    // vertex
673                    Bi::BaseInstance => "base_instance",
674                    Bi::BaseVertex => "base_vertex",
675                    Bi::ClipDistance => "clip_distance",
676                    Bi::InstanceIndex => "instance_id",
677                    Bi::PointSize => "point_size",
678                    Bi::VertexIndex => "vertex_id",
679                    // fragment
680                    Bi::FragDepth => "depth(any)",
681                    Bi::PointCoord => "point_coord",
682                    Bi::FrontFacing => "front_facing",
683                    Bi::PrimitiveIndex => "primitive_id",
684                    Bi::SampleIndex => "sample_id",
685                    Bi::SampleMask => "sample_mask",
686                    // compute
687                    Bi::GlobalInvocationId => "thread_position_in_grid",
688                    Bi::LocalInvocationId => "thread_position_in_threadgroup",
689                    Bi::LocalInvocationIndex => "thread_index_in_threadgroup",
690                    Bi::WorkGroupId => "threadgroup_position_in_grid",
691                    Bi::WorkGroupSize => "dispatch_threads_per_threadgroup",
692                    Bi::NumWorkGroups => "threadgroups_per_grid",
693                    // subgroup
694                    Bi::NumSubgroups => "simdgroups_per_threadgroup",
695                    Bi::SubgroupId => "simdgroup_index_in_threadgroup",
696                    Bi::SubgroupSize => "threads_per_simdgroup",
697                    Bi::SubgroupInvocationId => "thread_index_in_simdgroup",
698                    Bi::CullDistance | Bi::ViewIndex | Bi::DrawID => {
699                        return Err(Error::UnsupportedBuiltIn(built_in))
700                    }
701                };
702                write!(out, "{name}")?;
703            }
704            Self::Attribute(index) => write!(out, "attribute({index})")?,
705            Self::Color {
706                location,
707                blend_src,
708            } => {
709                if let Some(blend_src) = blend_src {
710                    write!(out, "color({location}) index({blend_src})")?
711                } else {
712                    write!(out, "color({location})")?
713                }
714            }
715            Self::User {
716                prefix,
717                index,
718                interpolation,
719            } => {
720                write!(out, "user({prefix}{index})")?;
721                if let Some(interpolation) = interpolation {
722                    write!(out, ", ")?;
723                    interpolation.try_fmt(out)?;
724                }
725            }
726            Self::Resource(ref target) => {
727                if let Some(id) = target.buffer {
728                    write!(out, "buffer({id})")?;
729                } else if let Some(id) = target.texture {
730                    write!(out, "texture({id})")?;
731                } else if let Some(BindSamplerTarget::Resource(id)) = target.sampler {
732                    write!(out, "sampler({id})")?;
733                } else {
734                    return Err(Error::UnimplementedBindTarget(target.clone()));
735                }
736            }
737        }
738        write!(out, "]]")?;
739        Ok(())
740    }
741}

743impl ResolvedInterpolation {
744    const fn from_binding(interpolation: crate::Interpolation, sampling: crate::Sampling) -> Self {
745        use crate::Interpolation as I;
746        use crate::Sampling as S;
747
748        match (interpolation, sampling) {
749            (I::Perspective, S::Center) => Self::CenterPerspective,
750            (I::Perspective, S::Centroid) => Self::CentroidPerspective,
751            (I::Perspective, S::Sample) => Self::SamplePerspective,
752            (I::Linear, S::Center) => Self::CenterNoPerspective,
753            (I::Linear, S::Centroid) => Self::CentroidNoPerspective,
754            (I::Linear, S::Sample) => Self::SampleNoPerspective,
755            (I::Flat, _) => Self::Flat,
756            _ => unreachable!(),
757        }
758    }
759
760    fn try_fmt<W: Write>(self, out: &mut W) -> Result<(), Error> {
761        let identifier = match self {
762            Self::CenterPerspective => "center_perspective",
763            Self::CenterNoPerspective => "center_no_perspective",
764            Self::CentroidPerspective => "centroid_perspective",
765            Self::CentroidNoPerspective => "centroid_no_perspective",
766            Self::SamplePerspective => "sample_perspective",
767            Self::SampleNoPerspective => "sample_no_perspective",
768            Self::Flat => "flat",
769        };
770        out.write_str(identifier)?;
771        Ok(())
772    }
773}
