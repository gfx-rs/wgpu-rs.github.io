Struct CommandAllocator

Source 
pub(crate) struct CommandAllocator {
    free_encoders: Mutex<Vec<Box<dyn DynCommandEncoder>>>,
}
Expand description

A pool of free wgpu_hal::CommandEncoders, owned by a Device.

Each encoder in this list is in the “closed” state.

Since a raw CommandEncoder is itself a pool for allocating raw CommandBuffers, this is a pool of pools.
