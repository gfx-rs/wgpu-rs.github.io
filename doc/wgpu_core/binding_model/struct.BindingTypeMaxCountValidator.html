pub(crate) fn validate_binding_arrays( &self, ) -> Result<(), CreateBindGroupLayoutError>

Validate that the bind group layout does not contain both a binding array and a dynamic offset array.

This allows us to use UPDATE_AFTER_BIND on vulkan for bindless arrays. Vulkan does not allow UPDATE_AFTER_BIND on dynamic offset arrays. See https://github.com/gfx-rs/wgpu/issues/6737
